# **App Name**: Academia Flow

## Core Features:

- Intelligent Analytics Dashboard: Visual overview of student demographics and enrollment statistics by course and year using real-time charts.
- Unified Student Directory: Advanced searchable directory with multi-tier filtering for course, gender, and enrollment year with server-side pagination.
- Smart Record Management: Full CRUD functionality for student profiles including an automated admission number generator tool.
- AI Profile Summarization: A tool that uses generative AI to analyze student performance and profile data to generate concise biographical summaries.
- Document & Media Vault: Secure upload system for student photographs with support for JPEG/PNG and integrated cloud storage linking.
- Institutional Activity Ledger: Comprehensive audit log that tracks record creations, updates, and deletions for accountability.
- Dynamic Responsive Forms: Intelligent multi-part forms with real-time validation for email, mobile, and DOB logic.

## Style Guidelines:

- The palette is built on 'Scholastic Authority'. Primary color: Deep Academic Indigo (#2a4e96). Background color: A refined, clean porcelain white with a blue undertone (#f1f3f8). Accent color: A vibrant cerulean (#1ab5e5) used for actionable highlights.
- A font pairing of 'Alegreya' for serif headlines to evoke an intellectual feel, and 'Inter' for sans-serif body text to maintain an objective and modern look.
- Monolinear, thin-stroke icons that emphasize precision and administrative clarity, keeping the UI looking clean and professional.
- A structured, grid-heavy dashboard layout utilizing 'bento-box' style cards to group related statistical and form data clearly.
- Micro-interactions on hover and subtle slide-in transitions for sidebars and modal dialogs to provide feedback without distraction.
